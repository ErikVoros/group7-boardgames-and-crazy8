# Group 7 Boardgames and Crazy 8

## Team
- Student 1: frontend for board game rental
- Student 3: Crazy 8 game logic and AI

## Project
This project combines:
- a board game rental web frontend
- a Crazy 8 game against the computer

## Structure
- `src/app` → frontend
- `src/game` → Crazy 8 game logic
- `tests` → test pages
- `docs` → project documentation

## Run
Use a local server to run the project.

Example:
python -m http.server 8000

or

https://erikvoros.github.io/group7-boardgames-and-crazy8/
